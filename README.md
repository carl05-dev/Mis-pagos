# Mis Pagos - Control Financiero Personal 📊

Una herramienta ligera, rápida y privada para el seguimiento de ingresos, bonos y tasas de cambio. Diseñada específicamente para funcionar como una **PWA (Progressive Web App)** en dispositivos móviles (iOS y Android).

## 🚀 Características
- **Seguimiento Detallado:** Registra sueldos, cestatickets, utilidades, prestaciones y vacaciones.
- **Conversión Automática:** Calcula montos en Bolívares y Dólares (BCV y Paralelo) usando tasas mensuales configurables.
- **Visualización:** Gráficos estadísticos mensuales integrados.
- **Privacidad Total:** Los datos se guardan localmente en el dispositivo (`localStorage`). No hay servidores externos.
- **Respaldo:** Permite exportar e importar toda la información en formato CSV.
- **Modo Mobile-First:** Interfaz optimizada para pantallas táctiles y teclado numérico.

## 📱 Instalación en el Celular
1. Sube el archivo `index.html` a tu hosting o ábrelo directamente en el navegador.
2. En iPhone (Safari): Presiona **Compartir** -> **Agregar a la pantalla de inicio**.
3. En Android (Chrome): Presiona los **tres puntos** -> **Instalar aplicación**.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Variables modernas y Safe Areas para iPhone).
- JavaScript (Vanilla JS).
- Chart.js para las gráficas.
